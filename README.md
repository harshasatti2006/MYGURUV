MYGURUV is a web-based application that allows students to vote for their favorite professors in a simple, transparent, and user-friendly manner.

The application is designed to encourage student participation and provide meaningful feedback in an organized way.

## Features
- Students can vote for their favorite professors
- Simple and clean user interface
- Secure and structured voting process
- Real-time vote tracking
- Easy to use for both students and administrators

## Technologies Used
- Python (Flask)
- HTML
- CSS
- JavaScript
- MySQL (if used)

## How to Run the Project
1. Install Python on your system
2. Install required packages using `pip install flask`
3. Run the application using `python app.py`
4. Open a browser and go to `http://127.0.0.1:5000`

## Purpose of the Project
This project is developed to understand web application development, user interaction, and database handling while solving a real-world problem in educational institutions.
